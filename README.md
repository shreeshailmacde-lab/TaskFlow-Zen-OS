# 🌊 TaskFlow Zen OS
> "A visual frequency for your most productive self."

TaskFlow Zen OS is a professional-grade timeboxing application designed for those who treat time like a craft. Unlike standard calendars, Zen OS uses a **24-hour Habit Pulse** to audit every minute of your day with digital precision.

![TaskFlow Screenshot](https://picsum.photos/seed/productivity/1200/600)

## ✨ Key Features

- **Zen Grid Interface**: A minimalist geometric calendar designed for focus.
- **Habit Pulse**: Real-time visual audit of your daily time utilization.
- **AI Motivation**: Dynamic productivity quotes generated to match your current progress.
- **Cloud Sync**: Secure Google Authentication and Firestore real-time database.
- **Task Infographics**: Visual icons for tasks including Gym, Work, Meditation, and more.
- **Smart Recurrence**: Support for daily, weekly, and custom repeating tasks.

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, Lucide Icons, Framer Motion (animations)
- **Charts**: Recharts
- **Backend**: Firebase (Auth & Firestore)
- **State Management**: React Hooks & Context API

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- A Firebase Project (for Auth and Database)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/taskflow-zen-os.git
   cd taskflow-zen-os
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env` file in the root directory and add your Firebase credentials:
   ```env
   VITE_FIREBASE_API_KEY=your_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   VITE_FIREBASE_APP_ID=your_app_id
   VITE_FIREBASE_DATABASE_ID=(default)
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```
